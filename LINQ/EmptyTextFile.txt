
LINQ - Language Integrated Queries
Запросы средствами языка (программирования)

Есть две формы LINQ: запросы / методы (расширения)

						// using System.Linq
Users.					// LINQ применяется к коллекции (IEnumerable)
Where(					// Методы принимают делегаты (лямбды)
u => 					// Параметры метода есть элементы коллекции (User)
u.Login == userLogin	// предикат - разновидность функции булевого типа
)						// Методы LINQ возвращают новую коллекцию
.Count()				// Ряд методов - терминальные, возвращают значение
